## 基础规则

快速复制:
```
{"popup_rules":
    [
        {"id":"gift_img","action":"gift_close"}
        {"id":"打开系统通知","action":"iv_close"}
    ]
}
```
详细说明：
- [{"id":"gift_img","action":"gift_close"}](#idgift_imgactiongift_close)
- [{"id":"打开系统通知","action":"iv_close"}](#id打开系统通知actioniv_close)
