快速复制:
```
{"popup_rules":
    [
        {"id":"tv_notify_title","action":"iv_close"},
        {"id":"签到&天&获得&车币","action":"GLOBAL_ACTION_BACK"},
        {"id":"开启消息推送","action":"iv_close"}
    ]
}
```
