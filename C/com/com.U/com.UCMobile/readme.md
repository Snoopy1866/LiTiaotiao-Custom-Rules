## 基础规则

快速复制:
```
{"popup_rules":
    [
        {"id":"开启消息通知","action":"GLOBAL_ACTION_BACK"}
    ]
}
```
详细说明：
- [{"id":"开启消息通知","action":"GLOBAL_ACTION_BACK"}](#id开启消息通知actionglobal_action_back)

### {"id":"开启消息通知","action":"GLOBAL_ACTION_BACK"}
