# com.kugou.android.lite（酷狗概念版）

## 基础规则

快速复制:
```
{"popup_rules":
    [
        {"id":"开启青少年模式","action":"知道了"},
        {"id":"ab7","action":"d9y"},
        {"id":"e7x","action":"ab2"}
    ]
}
```
详细说明：
- [{"id":"开启青少年模式","action":"知道了"}](#id开启青少年模式action知道了)
- [{"id":"ab7","action":"d9y"}](#idab7actiond9y)
- [{"id":"e7x","action":"ab2"}](#ide7xactionab2)

### {"id":"开启青少年模式","action":"知道了"}
关闭青少年模式弹窗

![](./assets/青少年模式弹窗.jpg)

### {"id":"ab7","action":"d9y"}
关闭广告弹窗

⚠ 仅测试 3.3.5 版本，其他版本可用性未知

![](./assets/广告弹窗.jpg)

### {"id":"e7x","action":"ab2"}
关闭 “我的” 页面右下角悬浮人物

⚠ 仅测试 3.3.5 版本，其他版本可用性未知

![](./assets/“我的”页面右下角悬浮人物.jpg)
