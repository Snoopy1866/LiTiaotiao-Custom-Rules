## 基础规则

快速复制:
```
{"popup_rules":
    [
        {"id":"青少年模式","action":"我知道了"},
        {"id":"你可能感兴趣的人","action":"关闭"}

    ]
}
```
详细说明：
- [{"id":"青少年模式","action":"我知道了"}](#id青少年模式action我知道了)
- [{"id":"你可能感兴趣的人","action":"关闭"}](#id你可能感兴趣的人action关闭)
