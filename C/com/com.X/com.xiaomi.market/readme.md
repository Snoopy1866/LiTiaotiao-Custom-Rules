## 基础规则

快速复制:
```
{"popup_rules":
    [
        {"id":"openAlertText","action":"btn_close_update_setting"},
        {"id":"=关闭","action":"=关闭"}
    ]
}
```
详细说明：
- [{"id":"openAlertText","action":"btn_close_update_setting"}](#idopenalerttextactionbtn_close_update_setting)
- [{"id":"=关闭","action":"=关闭"}](#id关闭action关闭)

### {"id":"openAlertText","action":"btn_close_update_setting"}
关闭 “开启更新提醒” 提示信息

![](./assets/开启更新提醒提示信息.jpg)

### {"id":"=关闭","action":"=关闭"}
