## 普通规则

快速复制:
```
{"popup_rules":
    [
        {"id":"single_banner","action":"close"},
        {"id":"=广告","action":"closeGray"}
    ]
}
```
详细说明：
- [{"id":"single_banner","action":"close"}](#idsingle_banneractionclose)
- [{"id":"=广告","action":"closeGray"}](#id广告actionclosegray)

### {"id":"single_banner","action":"close"}
关闭公告板弹窗

![](./assets/公告板弹窗.jpg)

### {"id":"=广告","action":"closeGray"}
关闭评论区广告信息
