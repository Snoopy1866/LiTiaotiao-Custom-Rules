## 基础规则

快速复制:
```
{"popup_rules":
    [
        {"id":"立即提现","action":"GLOBAL_ACTION_BACK"},
        {"id":"马上领取","action":"残忍退出"}
    ]
}
```

- [{"id":"立即提现","action":"GLOBAL_ACTION_BACK"}](#id立即提现actionglobal_action_back)
- [{"id":"马上领取","action":"残忍退出"}](#id马上领取action残忍退出)
