快速复制:
```
{"popup_rules":
    [
        {"id":"iv_bg","action":"iv_close"},
        {"id":"iv_ad","action":"tv_close_ad"}
    ]
}
```

- [{"id":"iv_bg","action":"iv_close"}](#idiv_bgactioniv_close)
- [{"id":"iv_ad","action":"tv_close_ad"}](#idiv_adactiontv_close_ad)
