# com.tencent.easyearn（企鹅汇图）

## 基础规则

快速复制:
```
{"popup_rules":
    [
        {"id":"青少年守护","action":"已满18周岁"},
        {"id":"打开通知","action":"取消"}
    ]
}
```
详细说明：
- [{"id":"青少年守护","action":"已满18周岁"}](#id青少年守护action已满18周岁)
- [{"id":"打开通知","action":"取消"}](#id打开通知action取消)

### {"id":"青少年守护","action":"已满18周岁"}
关闭青少年模式弹窗

![](./assets/青少年模式弹窗.jpg)

### {"id":"打开通知","action":"取消"}
关闭打开通知弹窗
