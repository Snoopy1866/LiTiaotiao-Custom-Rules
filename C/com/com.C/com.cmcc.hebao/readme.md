# com.cmcc.hebao（和包）

## 普通规则

快速复制:
```
{"popup_rules":
    [
        {"id":"iv_head","action":"返回首页"},
        {"id":"允许地理位置权限","action":"不允许"}
    ]
}
```
详细说明：
- [{"id":"iv_head","action":"返回首页"}](#idiv_headaction返回首页)
- [{"id":"允许地理位置权限","action":"不允许"}](#id允许地理位置权限action不允许)

### {"id":"iv_head","action":"返回首页"}
关闭首页广告弹窗

### {"id":"允许地理位置权限","action":"不允许"}
