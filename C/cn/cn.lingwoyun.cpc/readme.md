# cn.lingwoyun.cpc（领沃云电脑）

⚠ 该 App 使用了 优量汇 SDK 向用户提供广告弹窗，部分广告无法通过规则关闭。

## 基础规则

快速复制:
```
{"popup_rules":
    [
        {"id":"新人专属月卡","action":"GLOBAL_ACTION_BACK"}
    ]
}
```
详细说明：
- [{"id":"新人专属月卡","action":"GLOBAL_ACTION_BACK"}](#id新人专属月卡actionglobal_action_back)

### {"id":"新人专属月卡","action":"GLOBAL_ACTION_BACK"}
关闭新人专属月卡弹窗
